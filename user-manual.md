# GNC Group Invoice Extractor - User Manual

## What is the Invoice Extractor?
The Invoice Extractor automatically reads invoice images and converts them into organized data tables. Upload your invoice images, and the system extracts all the important information into structured categories.

## How to Access
1. Go to the application website
2. Click "Continue with Google"  
3. Use your @gncgroup.ca email account
4. Access the main dashboard

## How to Use the System

### Step 1: Upload Images
1. Click "Invoice Extractor" from the dashboard
2. Upload invoice images by:
   - Dragging files onto the upload area
   - Clicking to browse and select files
3. Supported formats: JPEG, PNG, GIF, BMP (up to 50MB each)

### Step 2: Choose Settings

**AI Model Selection:**
- **Gemini 2.0 Flash** (Default) - Fast and accurate
- **Gemini 2.5 Flash** - Faster with better accuracy
- **Gemini 2.5 Pro** - Best accuracy for complex documents

**Add Instructions:**
- Tell the system what information to extract
- Example: "Extract all items with description, quantity, and prices"

### Step 3: Review Images
- Check uploaded images are correct
- Assign page numbers to each image
- Remove unwanted images
- Ensure images are clear and readable

### Step 4: Process Documents
1. Click "Start Processing"
2. Watch the progress bar
3. Wait for completion

### Step 5: View Results
Your data appears in organized categories and tables.

## App Features

### Data Categories
The system organizes extracted information into:
- **Labour** - Work costs and time
- **Material** - Physical items and supplies  
- **Equipment** - Tools and machinery costs
- **Consumables** - Disposable items
- **Subtrade** - Subcontractor services
- **Labour Timesheet** - Time tracking records
- **Equipment Log** - Equipment usage records

### Result Views

**Category Tables:**
- Each category shows extracted items
- Description, quantity, unit price, total amount
- Page reference for each item

**Consolidated View:**
- All categories combined in one table
- Grand total across all categories
- Cross-reference to source pages

**Summary Totals:**
- Total amount for each category
- Item count for each category
- Overall grand total

### Page Reference Feature
- Click "View" buttons next to any data
- See the original image where data was found
- Verify extracted information is correct

### Real-Time Processing
- Watch progress as images are processed
- See status for each page
- Error indicators if issues occur

## Using the Features

### Upload Multiple Images
- Select multiple files at once
- Drag and drop multiple images
- Process batches of invoices together

### Custom Instructions
- Write specific extraction requests
- Target particular data fields
- Improve accuracy with detailed instructions

### Model Selection
- Choose faster processing for simple documents
- Use advanced models for complex layouts
- Balance speed vs accuracy based on needs

### Data Verification
- Use page view buttons to check accuracy
- Compare extracted data with original images
- Verify totals and calculations

### Category Organization
- Data automatically sorted into relevant categories
- Easy navigation between different data types
- Clear separation of costs and timesheet data

This system makes invoice processing simple - upload images, let AI extract the data, and get organized tables with all your invoice information.
